# `nuxi init`

```{bash}
npx nuxi init|create [--verbose|-v] [--template,-t] [dir]
```
`init` 命令 初始化一个新的Nuxt项目。

Option        | Default          | 描述
-------------------------|-----------------|------------------
`--verbose, -v` | `false` | 输出安装过程中的信息。
`--template, -t` | `nuxt/starter#v3` | 指定模版的git仓库
`dir` | `nuxt-app` | 安装陌路的名称
