# Layer0

Nitro 提供了一个内置的预设，来生成[Layer0](https://www.layer0.co/) 兼容的输出格式


Layer0 扩展了传统CDN 的功能，不仅托管你的静态内容，还为渐进式Web应用程序提供服务器端渲染。并在网络边缘缓存你的API和HTML，为你的用户提供最快的流量体验。
