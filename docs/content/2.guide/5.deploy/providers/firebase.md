# Firebase Hosting

如何把Nuxt部署到 [Firebase Hosting](https://firebase.google.com/docs/hosting).

Nitro 支持 [Firebase Hosting](https://firebase.google.com/docs/hosting) 云函数开箱即用.

**注意**: 要将Nitro与云函数功能结合使用，你需要 **B 计划**
