# Gitignore 文件

`.gitignore` 文件中指定git 不需要跟踪的文件。在[git文档](https://git-scm.com/docs/gitignore)中了解更多.

我们建议 `.gitignore`文件**至少包含**下列条目的文件：

```bash [.gitignore]
# Nuxt dev/build outputs
.output
.nuxt
# Node dependencies
node_modules
# System files
*.log
```
