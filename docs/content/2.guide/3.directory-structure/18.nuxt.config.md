# Nuxt 配置文件


Nuxt 可以使用一个 `nuxt.config` 来轻松配置，它的扩展名可以是 `.js` `.ts` `.mjs`

```ts
import { defineNuxtConfig } from 'nuxt'

export default defineNuxtConfig({
  // My Nuxt config
})
```

为了确保你的配置是最新的，当检测到nuxt的主要配置文件（`.env` `.nuxtignore` `.nuxtrc` dot 文件）更新时，Nuxt会完全重新启动
